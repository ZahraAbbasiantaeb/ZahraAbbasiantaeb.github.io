Selected Academic Projects

Deep Learning | M.Sc.

Deep Feed-Forward Neural Network for Classifying Mnist Dataset

In this project various optimizers and cost functions, and different values for batch size, network depth, and network width is tested for finding the best model. Accomplished using Estimator package of tensorflow.

EBF and RBF Neural Networks for Classification, Function Approximation, and Time Series Prediction

In this project different number of centers and noise effect is experimented for classifying Swiss Roll dataset, approximating sin function, and predicting daily total female births in California(1959). Accomplished using tensorflow.

SOM Neural Network for Clustering and Dimensionality Reduction of High-Dim dataset

In this project different neighbourhood topologies are implemented. Best clustering model is found by experimenting various neighbourhood topologies and different values for neighbourhood radious. Also, dimensionality of High-Dim dataset is reduced by PCA and SOM and these two were compared by feeding the results to a feed-forward neural network for classification. Accomplished using tensorflow.

Deep Convolutional Neural Network for Classifying STL dataset and Classifying CIFAR dataset using Transfer Learning

In this project various Ppooling strategies and different values for dropout rate, kernel layers, convolution layers, and kernel size were experimented for finding the best architecture. Also ResNet network was fine-tuned for classifying CIFAR dataset. Accomplished using Estimator package of Tensorflow and keras.

Recurrent Neural Network for Stock Market Prediction

In this project various recurrent units like LSTM, GRU, BiLSTM, and BiGRU and their stacked form with different count of layers are tested. Finally the best predictor for each stock is founded. Accomplished using keras.

Convolutional Generative Adversarial Network (GAN) for Simpsons-faces dataset

In this project a convolutional model for adversarial and a de-convolutional model for generative model are implemented. Accomplished using keras.

Neural Machine Translation with Encoder-Decoder Architecture for Spanish-English and Persian-English datasets

In this project an encoder-decoder architecture is implemented for machine translation. Various recurrent units with simple and stacked form are tested for finding the best model. Accomplished using keras.

Natural Language Processing | M.Sc.

Hierarchical Attention-based Question Answering (Text-based) for Wiki-QA dataset

In this project two separate encoders are used for encoding the question and the candidate answer sentences. Two different attention layers are implemented in this project: cross-attention and self-attention. Glove pre-trained word embeddings are also used. This model computes the relevance score for each question and answer pair. Accomplished using keras.

Text Classification with Language Model for Hamshahri corpus

In this project unigram and bigram models are implemented for text classification and Absolute Discounting method is implemented for smoothing. Features are extracted using information gain algorithm and classification is tested by using just these features. Best value for delta parameter in absolute discounting is found by experimenting different values. Accomplished using python.

Text Clustering and Representation for Hamshahri corpus

In this project representation of each document is obtained in four different ways: 1) averaging word2vec representations of the document's words. 2) TF-IDF weighted averaging of word2vec representations of the document's words. 3) Doc2vec Representation. 4) Reduce the dimensionality of the doc-word matrix, which is filled with TF, by PCA. Then documents are clustered using the Kmeans algorithm. Accomplished using Scikit, NLTK, and Gensim Libraries in python.

Convex Optimization | M.Sc.

Implementation of BFGS, Newton, Steepest Descent, and Trust Region Algorithms

For finding the trust region Cauchy Point and Dog Leg methods are implemented. All of the mentioned algorithms are used for optimizing Rosenbrock function. Accomplished using python.
